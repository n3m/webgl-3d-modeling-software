# webgl-3d-modeling-software
A Basic implementation of a 3D Modeling Software in WebGL

- Features:
  - Create any kind of surfaces with all the vertices that you need.
  - Rotate your Model
  - Translate your Model
  - Scale your Model
- Create multiple models!
